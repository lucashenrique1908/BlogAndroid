BlogAndroid

This repository hosts the BlogAndroid project, a static blog page created to practice front-end development and layout design.

Live version:
https://lucashenrique1908.github.io/BlogAndroid/

Description

BlogAndroid is a responsive blog template focused on Android-related content.
It presents a structured layout with navigation, articles, images, and styled text, aiming to simulate a real blog interface.

This project was developed mainly for training and learning purposes.

Technologies Used

 - HTML

 - CSS

Purpose

* The main goals of this project are to:

* Practice semantic HTML

* Improve CSS styling and layout organization

* Build a static blog-style webpage

The project does not include backend functionality or a CMS and is intended for educational and portfolio use.

How to Run Locally

Clone the repository: git clone https://github.com/lucashenrique1908/BlogAndroid.git

Open the index.html file in a modern web browser.

Notes

This project represents part of the continuous development of front-end fundamentals, focusing on structure, responsiveness, and visual organization.
